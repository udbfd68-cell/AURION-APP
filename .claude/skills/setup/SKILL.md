# Vercel Setup

## Install CLI

```bash
npm install -g vercel
```

## Authenticate

```bash
vercel login
```

## Link Project

```bash
vercel link
```

## Environment Variables

```bash
vercel env ls
vercel env add <NAME>
```

## Configuration

Check for `vercel.json` and `.vercelignore` files.
